# Maypole

Place-based messaging app

Maypole is a place-based messaging app built with Flutter with MVVM architecture and Riverpods, Firebase, and the Google Places API. It
allows users to chat within threads attached to physical locations in Google Maps, and DM users they
meet there. Ads are served within the app using Google AdMob. Maypole is available on Android, iOS, web, MacOS, and Linux.
